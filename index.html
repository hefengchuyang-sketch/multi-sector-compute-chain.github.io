Project Architecture

1. Order Layer

Creates transparent compute orders with deterministic pricing and explicit constraints.

2. Execution Layer

Routes workloads to miners, tracks runtime status, and stores verifiable execution evidence.

3. Settlement Layer

Calculates miner payout, platform fee, and treasury allocation with fully visible outputs.
