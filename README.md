# Project Hyperion Showcase Blog

一个可本地运行、可直接上传 GitHub 的炫酷科幻风项目介绍博客，适合用于奖学金申请展示。

## 本地运行

方式 1：直接双击打开 `index.html`。

方式 2：使用本地静态服务器（推荐）：

```powershell
cd thiel-scholarship-blog
py -3 -m http.server 8080
```

然后访问：

- http://127.0.0.1:8080

## 内容结构

- Hero：项目愿景
- 使命：问题、方案、价值
- 进展：核心指标
- 博客：迭代记录（可扩展）
- 路线图：12 个月目标
- 申请导向信息：便于奖学金材料整理

## 快速改成你的信息

编辑：

- `index.html`（标题、叙事、路线图）
- `script.js`（博客条目）
- `styles.css`（视觉风格）

## 部署建议

可直接部署到 GitHub Pages。
